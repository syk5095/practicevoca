• 영단어 (어원편)>

  • 최중요 어근

  • cap

    잡다(hold), (생각을)취하다(take)

  • 변화형:cip, ceice, cept


  • capable : (동)~할 수 있는, 유능한[cap(잡다)+able(할 수 있는)]

  • capture : (동)포회하다, 사로잡다, (명)생포, 포획, 억류[cap(잡다)+ture(명접)]

  • anticipate : (동)예상하다, 예측하다[ante/i(먼저)+cip(취하다)+ate(동접)]

  • participate : (동)참여하다, 참가하다[part(부분)+cip(잡다)+ate(동접)]

  • conceive : (동)(생각을)품다, 생각하다, 임신하다[con(완전히)+ceive(취하다)]

  • deceive : (동)속이다, 기만하다, 사기 치다[de(떨어져)+ceive(취하다)]

  • perceive : (동)인식하다, 감지하다[per(완전히)+ceive(잡다)]

  • receive : (동)받다, 받아들이다[re(다시)+ceive(잡다)]

  • accept : (동)수락하다, 받아들이다[ac(~쪽으로(ad))+cept(취하다)]

  • contraception : (명)피임[contra(~에 반대하여)+cept(취하다)+ion(명접)]

  • except : (전,접)~을 제외하고는[ex(밖으로)+cept(취하다)]

  • intercept : (동)가로막다, 가로채다[inter(사이에)+cept(취하다)]

  • susceptible : (형)영향을 받기 쉬운, 민감한[sus(아래에서)+cept(잡다)+ible(할 수 있는)]
